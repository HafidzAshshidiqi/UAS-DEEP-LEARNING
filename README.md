# IoT Anomaly Detection using CNN-LSTM

This project implements a hybrid CNN-LSTM model for anomaly detection
on IoT temperature sensor data.

## Dataset
IoT Temperature Sensor Dataset (Kaggle)
IOT-temp.csv

## Method
- CNN for feature extraction
- LSTM for temporal dependency modeling

## Evaluation
- Accuracy
- Precision, Recall, F1-score
- ROC-AUC
- Baseline comparison (SVM, Autoencoder)

## Deployment
Model is converted to TensorFlow Lite for edge deployment.
